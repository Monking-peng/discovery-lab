# HelpHub seed study

All files in this directory are synthetic and safe for a public portfolio demo.

The dataset intentionally contains:

- frequently mentioned low-risk auto-reply requests;
- fewer but higher-impact enterprise escalation failures;
- repeated tickets from the same account;
- conflicting cohort needs;
- a prompt-injection string embedded as source data;
- evidence that supports human-assisted escalation, not autonomous escalation.

The expected product insight is not preloaded as a fact. The system must preserve the distinction between source text, extracted observation, interpretation, and final human-approved decision.
